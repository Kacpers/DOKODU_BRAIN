## FILOZOFIA SYSTEMU

1. **Capture first, organize later** ‚Äî najpierw wrzuc do Inboxu, nie trwa kategoryzowac w locie
2. **Projects die without next actions** ‚Äî kazdy projekt musi miec "Nastepny krok" z deadline
3. **Resources compound** ‚Äî biblioteka promptow i blueprintow rosnie z kazdym projektem
4. **Weekly Review is non-negotiable** ‚Äî bez piatku caly system sie sypie
5. **One source of truth** ‚Äî jezeli informacja jest w 2 miejscach, jest w zadnym
