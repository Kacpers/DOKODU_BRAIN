### Struktura

**H1:** Cursor Pro dla Studentów w Polsce — Darmowy Dostęp, Weryfikacja i Co Robić gdy Się Nie Uda

**Intro:** (150 słów)
- Hook: program studencki istnieje, ale ma pułapkę dla Polski
- Co artykuł odpowiada: jak się zarejestrować, co zrobić gdy .edu.pl nie działa, jakie są alternatywy
- Dla kogo: student polskiej uczelni, uczestnik bootcampu

**H2: Czy Cursor Pro jest Darmowy dla Studentów?** (~250 słów)

- TAK — Cursor ma program GitHub Student Developer Pack (przez SheerID weryfikację)
- Plan "Hobby" jest darmowy dla każdego (50 requests/mies.)
- Student Education Plan: pełne Pro bez oplat (podobnie jak GitHub Student Pack)
- Wymogi: aktywny status studenta + uczelniany email lub dokument

**H2: Jak Aktywować Cursor Pro Student — Krok po Kroku** (~300 słów)

1. Idź na cursor.com/students
2. Kliknij "Apply for Student Access"
3. SheerID poprosi o email uczelniany (.edu lub .edu.pl)
4. Weryfikacja: automatyczna (email) lub manualna (załaduj legitymację)
5. Oczekiwanie: 1-3 dni robocze na weryfikację manualną

**H3: Problem z Polskimi Adresami .edu.pl**
- SheerID NIE rozpoznaje automatycznie wszystkich domen .edu.pl
- Znane problemy: agh.edu.pl, pw.edu.pl, uw.edu.pl — różne wyniki
- Co zrobić: metoda alternatywna — załaduj legitymację studencką (skan/zdjęcie)
- Tip: jeśli email nie zadziałał, prześlij ISIC lub legitymację — to wystarczy
- Kontakt support: support@cursor.com z tematem "Student verification - Polish university"

**H2: GitHub Student Pack + Cursor — Czy to Działa?** (~200 słów)

- GitHub Student Developer Pack od 12 marca 2026 zmienił strukturę
- Cursor jest partnerem GitHub Education — dostęp przez pack.github.com
- Jak aktywować przez GitHub: pack.github.com → wyszukaj Cursor → "Get offer"
- To INNA ścieżka niż bezpośrednia — często lepsza, bo GitHub weryfikuje status oddzielnie
- GitHub akceptuje: .edu, .edu.pl i legitymację przez SheerID — wyższy success rate

**H2: Alternatywy dla Studentów — Darmowe Narzędzia AI do Kodu** (~350 słów)

Tabela (stan na marzec 2026):

| Narzędzie | Plan Studencki | Co Dostaniesz | Jak Aktywować |
|-----------|---------------|----------------|---------------|
| Cursor | Pro gratis | Pełny Cursor Pro | SheerID / GitHub Pack |
| GitHub Copilot | Individual gratis | Copilot Chat + autocomplete | GitHub Student Pack |
| Windsurf | Plan darmowy | 200 kredytów/mies. | Rejestracja zwykła |
| Codeium | Darmowy forever | Nieograniczony (starsze modele) | Rejestracja zwykła |
| JetBrains | Wszystkie IDE gratis | Full JetBrains Suite | JetBrains Education |
| Replit | Plan Hacker gratis | Replit AI + hosting | GitHub Student Pack |

- Rekomendacja dla studenta-beginnera: Codeium (zero ograniczeń, plug do VS Code)
- Rekomendacja dla studenta-zaawansowanego: Cursor Pro (najlepszy AI IDE)
- Stack stack: Cursor + GitHub Copilot = redundancja, wybierz jedno

**H2: Co Zrobić gdy Weryfikacja Się Nie Udaje?** (~200 słów)

1. Spróbuj ponownie z innym adresem email uczelni (jeśli masz kilka)
2. Prześlij legitymację studencką jako JPG/PDF
3. Skontaktuj się z support przez cursor.com/support
4. Alternatywnie: używaj GitHub Student Pack path (wyższy success rate)
5. Tymczasowo: plan Hobby jest darmowy zawsze — 50 requestów/mies. wystarczy do nauki
6. Jeśli za rok (po ukończeniu uczelni): przejdź na Hobby lub płatny Pro

**H2: Czy Warto Uczyć Się na Cursor jako Student?** (~150 słów)

- TAK — Cursor to standard w branży IT od 2024
- Firmy pytają o doświadczenie z AI-assisted coding na rozmowach
- Czas inwestycji: 1-2 tygodnie do biegłości podstawowej
- Rekomendacja: zacznij od Hobby, aktywuj studencki po weryfikacji
