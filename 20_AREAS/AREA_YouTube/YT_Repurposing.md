## Proces repurposingu

```
1. yt-dlp --write-auto-sub --sub-lang pl --skip-download "URL" -o /tmp/nazwa_filmu
2. python3 → wyczyść VTT → plain text (/tmp/transkrypcja.txt)
3. Przeczytaj transkrypcję → napisz artykuł (nie przepisuj! artykuł > transkrypcja)
4. W artykule dodaj sekcję "## Film: [tytuł]" z samym linkiem YT na osobnej linii
   → blog automatycznie zamieni link na odtwarzacz LiteYouTube
5. POST /api/external/posts jako draft (przez API 172.19.0.7:3000)
6. Dodaj do tabeli "Gotowe drafty" powyżej
7. Kacper: dodaj zdjęcie + publikuj
```

### Format sekcji z filmem (wklejaj do artykułu):
```markdown
## Film: [tytuł roboczy]

https://youtube.com/watch?v=VIDEO_ID

Poniżej znajdziesz pisemny tutorial — możesz go czytać bez oglądania,
albo używać jako notatki obok filmu.
```
