## STRUKTURA KURSU (moduły)

### Modul 0: Filozofia Automatyzacji (FREE — lead magnet)
- Czym jest n8n i dlaczego nie Zapier/Make?
- Jak myslec procesami, nie narzedziem
- Setup srodowiska (lokalny vs self-hosted)

### Modul 1: n8n Fundamenty (3h)
- Interfejs, przeplywy, triggery
- HTTP Request — twoj najwazniejszy node
- Autentykacja: OAuth2, API Keys, Basic Auth
- Error Handling: TryCatch, Error Workflow

### Modul 2: Dane i Transformacje (2h)
- JSON: czytanie, pisanie, manipulacja
- Code Node (JavaScript/TypeScript) — kiedy i jak
- Expressions i Jinja-like templating
- Praca z tablicami: Split, Merge, Aggregate

### Modul 3: Integracje Produkcyjne (4h)
- Google Workspace: Gmail, Sheets, Drive, Calendar
- Microsoft 365: Outlook, Teams, SharePoint
- CRM: HubSpot, Pipedrive (via API)
- Bazy danych: PostgreSQL, Airtable
- Komunikacja: Slack, Telegram Bot

### Modul 4: AI w n8n (3h) — PREMIUM
- AI Agent node — konfiguracja i prompty
- RAG: podlacz wiedze do agenta (Google Drive jako baza)
- Analiza PDF / email (Gemini + GPT-4o)
- Guardrails: kiedy AI mowi "nie"

### Modul 5: Bezpieczenstwo i Compliance (1.5h) — UNIKALNE
- Vault: przechowywanie sekretow
- RODO w automatyzacjach: co wolno, co nie
- Server-side tracking zamiast klienta
- Logowanie i audyt w n8n

### Modul 6: Sprzedaz i Delivery (2h) — AGENCJA FOCUS
- Jak wyceniac projekty automatyzacji
- Discovery Call: pytania, ktore musisz zadac
- Umowa IT: co musi sie w niej znalezc
- Dokumentacja i handover dla klienta
- Retainer: jak zbudowac pasywny przychod
